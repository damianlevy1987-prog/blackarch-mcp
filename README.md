## 🚀 Quick Start

### MCP Server
```bash
python3 blackarch_mcp_v2.py
```

### CLI
```bash
./blackarch.sh stats
./blackarch.sh search sqlmap
./blackarch.sh cheat scanner
```

### Web Interface
```bash
python3 blackarch_web.py
# Open http://localhost:8080
```

## 🔧 Installation

```bash
# Clone or download
git clone <repo>
cd blackarch-mcp

# Run installer
./install.sh

# Or use directly
python3 blackarch_mcp_v2.py
```
